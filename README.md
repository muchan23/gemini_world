# gemini_world

Interactive 3D Box World experience built with React Three Fiber.

A mysterious box floating in space opens to reveal a miniature cosmic universe inside. Click to dive in and explore the cosmos.

## Features

- Phase-based interaction: idle -> opening -> opened -> diving -> immersed
- 3D box with animated lid, glowing edges, and transparent walls
- Cosmic world with nebulae, planets, orbital rings, and particle streams
- Gemini logo particle effect that drops into the box
- Dynamic star field that expands when immersed
- Smooth camera animations between all phases

## Tech Stack

- React + TypeScript
- Three.js / React Three Fiber
- Vite

## Getting Started

```bash
npm install
npm run dev
```
